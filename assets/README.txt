# Banner Image Information

You should create a banner image for your project with the following specs:
- Recommended size: 1280x640 pixels
- Format: PNG or JPG
- Content: Include the project name "AI-Powered EDA & Feature Engineering Assistant" and relevant data visualization graphics
- Color scheme: Use a dark theme with bright accent colors for data visualizations

Once created, name it "banner.png" and place it in this directory.

You can use tools like Canva, Figma, or Adobe tools to create a professional banner.

The banner will be displayed at the top of your README.md file on GitHub.
